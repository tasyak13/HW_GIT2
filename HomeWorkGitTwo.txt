1. Как отказаться от изменений в файле (файлах) находящийся в рабочем каталоге (work directory)?
git checkout

2. Как вернуть файл из индекса в рабочий каталог?
git restore --staged <file>

3. Как отменить ошибочный commit?
git reset HEAD^ (^ - количество означает на сколько шагов назад мы хотим откатиться)
or
git reset HEAD~n (n - цифровое обозначение шагов, на которое мы хоти откатиться)

4. Как создать новую ветку и переключится в нее?
git chgit eckout -b NameOfBranch

5.В чем отличение между fast-forward merge and non fast-forfard merge?
fast-forward -просто сливает обе ветки, и всё что было сделано в другой ветке переносится в главную ветку
non fast-forfard merge - когда сливаем обе ветки, и есть разные изменения в одном файле, но на обеих ветках. При таком слияние попросит сделать ещё один commit

6. Когда может возникнуть конфликт слияния веток, как  его разрешить?
Если при обьединение двух веток происходит слияние двух файлов(при разьеденение веток это был один
и тот же файл), которые были изменены в обоих ветках. Открываем файл который надо обьеденить.
Изменяем нужные нам изменения в редакторе. Добовляем индекс и commit.(or git commit -a)

7. Можно ли удалить ветку? Как? 
git branch -d NameOfBranch
